# Space Novel: Forsaken Cargo - Официальный сайт

Сайт для публикации информации о вселенной Space Novel и игре Forsaken Cargo.

## Структура

- `index.html` - главная страница сайта
- `styles.css` - стили в ретро sci-fi стиле
- `script.js` - интерактивные функции
- `images/` - папка с изображениями персонажей и игровыми артами

## Запуск

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server
```

Затем откройте http://localhost:8000 в браузере.

## Особенности

- Ретро sci-fi дизайн с космическим фоном
- Адаптивная верстка для всех устройств
- Плавная прокрутка и анимации
- Ссылки на все социальные сети
- Информация об игре и вселенной
